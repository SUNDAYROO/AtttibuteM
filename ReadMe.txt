How to Use:

1. Enter the desired attribute name.

2. Select the objects you want to add the attribute to and click the "Select" button.

3. A set group will be created with the name you entered, and the attribute will be added.
 - The attribute will be initialized with a default float value of 1.0.
 - The attribute will be associated with the Shape node.

4.To make changes or remove an added attribute, select the objects you want to modify, enter the attribute's name, and click "Delete."
 - Ensure there is an existing attribute with the same name in the list.

5. If you wish to deselect objects and clear the window, simply click the "Clear" button, and all contents will be cleared.

Functionality: Selecting a parent group will automatically include its child objects.
To delete specific object attributes, select the object, input the attribute name you want to remove, and click "Delete."
Any spaces you enter will be naturally converted to underscores.
